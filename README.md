### Spotify dashboard (roster / overview)

The app uses the **Spotify Web API** with **client credentials** (`SPOTIFY_CLIENT_ID` + `SPOTIFY_CLIENT_SECRET`). **Spotify for Artists admin** does not authenticate this; you must create a [Developer app](https://developer.spotify.com/dashboard) and add both variables to **Vercel → Project → Settings → Environment Variables** for Production (then redeploy).

After deploy, open **`/api/spotify/health`** — `ok: true` means credentials work. Then **`/api/spotify?q=ArtistName`** should return JSON.

The dashboard shows **public** API data (followers, popularity, top tracks, etc.). It is not a full mirror of **Spotify for Artists** analytics (e.g. detailed streaming reports).
